# Support

## Where To Ask For Help

- Use a bug report issue when you have a reproducible defect
- Use a feature request issue when you have a clear product or workflow improvement in mind
- For setup questions, troubleshooting, or usage help, open an issue with enough detail to reproduce your environment

Include:

- operating system
- Rust version
- Qdrant version
- whether you are using `tei_shim`, `text-embeddings-inference`, or lexical-only mode
- logs or screenshots when relevant

## Security

Do not use public issues for security reports. Follow [SECURITY.md](SECURITY.md).
